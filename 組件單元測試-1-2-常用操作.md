# 組件單元測試-1-2-常用操作

### test 內使用 vue file 的 methods

```js
// 取得掛載的組件對象 wrapper.vm
// 就可以使用它裡面所有的屬性包括 methods
wrapper.vm.methodName()
```

### test 內改變 props data

```js
// 使用 async await
await wrapper.setProps({ foo: 'bar' })
```
